{{ _("Quick Start") }}

To register your samples with a new collection in the repository:

  1. Log in via (AAF)
  2. Submit a request to create a sample collection
  3. You will be notified via email after the collection is created
  4. Select your collection and choose:
    • Add Sample(register individual samples via the online form)
    • Start Batch Upload (register multiple samples by uploading a spreadsheet )

To register your samples with an existing collection in the repository:

  1. Log in via (AAF)
  2. Select your collection-of-interest
  3. Click
    • Add Sample(register individual samples via the online form)
    • Start Batch Upload (register multiple samples by uploading a spreadsheet )

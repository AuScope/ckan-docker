    {{ _('Referencing and Citing the AuScope Data Repository and Its Datasets') }}

    Please use the following format to acknowledge both the AuScope Data Repository and each individual dataset in any academic literature reports etc.

    Citing Repository

    • References: Devaraju A., Woodman S., Taherifar N., Motevalli Soumehsaraei B., Fazio V., Bradley S., Wyborn L., Klump J., Farrington R. (2024). AuScope Data Repository. https://www.re3data.org/repository/r3d100014363
    • In-text citation: “XX data from the AuScope Data Repository (Devaraju et al. 2024)…”
    • Acknowledgements: Data was obtained from the AuScope Data Repository (https://repository.data.auscope.org.au/) funded by AuScope and supported by the Australian Government through the National Collaborative Research Infrastructure Strategy (NCRIS).

    Citing Dataset

    The repository will automatically generate citations for each dataset it publishes using the Datacite Citation format. The citation is located on the metadata page of each dataset.

    • References: Graeme Beardsmore, Joe Hamad (2024): Gippsland and Murray Basin Fiber Optic Distributed Temperature Sensing Data. AuScope. (Dataset). https://doi.org/10.82669/meq3cwju
    • In-text citation: “The data showed the temperature range (Beardsmore et al. 2024)…”
    • Data Availability Statement: Datasets used in this study are available via the AuScope Data Repository (https://repository.data.auscope.org.au/). The datasets used are (Beardsmore et al. 2024); (Smith and Jones 2020); …

    *Note: The DOI (10.82669/meq3cwju) above is shown for demo only; therefore it is not actionable.

    {{ _('Referencing and Citing the AuScope Data Repository and Its Datasets') }}

    Please use the following format to acknowledge both the AuScope Data Repository and each individual dataset in any academic literature, reports, etc.

    Citing Repository

    • References: Devaraju, A., Woodman, S., Taherifar, N., Motevalli Soumehsaraei, B., Fazio, V., Bradley, S., Wyborn, L., Klump, J., Farrington, R. (2024), AuScope Data Repository. Registry of Research Data Repositories(re3data.org). http://doi.org/10.17616/R31NJNKR
    • In-text citation: “XX data from the AuScope Data Repository (Devaraju et al., 2024)…”
    • Acknowledgements: Data was obtained from the AuScope Data Repository (https://repository.data.auscope.org.au/), funded by AuScope and supported by the Australian Government through the National Collaborative Research Infrastructure Strategy (NCRIS).

    Citing Dataset

    The repository will automatically generate citations for each dataset it publishes using the Datacite Citation format. The citation is located on the metadata page of each dataset.

    • References: James R. Austin (2024): Cloncurry METAL: multimodal integrated mineral system characterisation data. AuScope. (Dataset). https://doi.org/10.60623/82trleue
    • In-text citation: “The data represents complex mineral system (Austin, JR., 2024)…”
    • Data Availability Statement: Datasets used in this study are available via the AuScope Data Repository (https://repository.data.auscope.org.au/). The datasets used are (Austin, JR 2024); (Smith and Jones 2020); …
